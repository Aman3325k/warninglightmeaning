# GEMINI.md — Gemini Agent Context for warninglightmeaning.com

## What This Project Is
warninglightmeaning.com is a static Astro.js website that explains
car dashboard warning lights to US drivers. It is monetized via
Google AdSense. It is deployed on Cloudflare Pages.

## Your Role
You are the design and content assistant for this project.
Antigravity handles all code. Claude handles all strategy.
You handle UI design (via Stitch) and content generation when asked.

## Rules You Must Follow
- Never suggest changing output: 'static' in astro.config.mjs
- Never suggest SPA routing or client-side navigation
- Always follow DESIGN.md for colors, fonts, and components
- Always follow ROUTES.md before suggesting any new URLs
- Always check MEMORY.md to know what is already built
- Always check TASKS.md to know what is next
- Never invent new features not listed in TASKS.md
- Never modify AGENTS.md, MEMORY.md, TASKS.md, ERRORS.md directly

## Design System Summary
- Colors: white #FFFFFF, charcoal #1A1A1A, amber #F59E0B
- Font: Inter
- Framework: Tailwind CSS v4
- No gradients, no stock photos, no decorative elements
- Urgency: Red = Stop Immediately, Amber = Check Soon, Gray = Info Only
